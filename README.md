# 2017 Educator Lessons

Teaching materials and lesson roadmap from 2017. Used as a map for student lessons (JavaScript, DOM, OOP, ES6).

---

## [→ Main roadmap (navigation)](ROADMAP.md)

Use the **main roadmap** to jump between lessons (01–38), help & reference, and ready works.

---

## Structure

- **`01_GiveLesson` … `38_GiveLesson`** — Numbered lesson folders (HTML/JS demos, tasks, homework). Each has a [ROADMAP](ROADMAP.md) for lesson-specific navigation.
- **`About_My_Lessons_HELP_FILES`** — Reference: BOM, DOM, OOP, ES6, RegExp, difficult tasks, JS exercises.
- **`ReadyWorksByJS`** — Ready-made examples: clocks, sliders, tables, carousels, calculators, etc.
- **`Cool_mini_projects_by_js`** — Mini projects: sliders, puzzle, zoom, loto, search.
- **`Study_Images`** / **`Images_by_Lessons`** — Lesson assets and screenshots.
- **`Modules_import_and_export`**, **`Must_Know_abot-JS`**, **`Interview Questions`**, **`EXAM`** — Extra topics.

## How to use

Open any `.html` file in a browser (local file or via a simple static server). No build step required.
